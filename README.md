# Pizza-Sales-Analysis
Excel Dashboard &amp; SQL Analysis of Pizza Sales Dataset

# Objective
To analyze pizza sales data and create an interactive Excel dashboard and SQL reports to identify KPIs, sales trends, and best/worst-selling pizzas.

# Project Files
- Excel Dashboard – Interactive dashboard showing KPIs, trends, and category insights.
- SQL Queries – KPI calculation, best/worst sellers, time analysis.
- Dataset – Pizza sales dataset used for the analysis.

# Tools Used
- Microsoft Excel
- SQL
- Pivot Tables, Charts, Slicers

# Key Insights

- Weekend sales showed higher order volume.
- Evening hours had peak sales.
- Classic category & Large size contributed maximum revenue.
